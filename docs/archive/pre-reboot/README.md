# Pre-reboot documentation archive

Archived 12 July 2026 when Mathland adopted the open-world Atlas–Studio–Observatory product architecture for motivated adult learners.

These files describe the previous application, its Module 2 release plan, Lesson v2 experiment, tier/template systems, and learner-comparison research. They remain useful as historical evidence and as a source of implementation ideas, but they are not active requirements.

The active doctrine is [../../PRODUCT_DOCTRINE.md](../../PRODUCT_DOCTRINE.md).

- `RELEASE_BASELINE.md` — previous Module 2 release baseline.
